# Weapons Store Project

مشروع متجر أسلحة إلكتروني متكامل مبني باستخدام PHP (PDO) و MySQL.

## المتطلبات
- **Web Server**: Apache (XAMPP/WAMP)
- **PHP**: 7.4+
- **Database**: MySQL 5.7+

## التثبيت
1. قم بنسخ مجلد `Store` إلى مجلد `htdocs` في XAMPP.
2. افتح المتصفح على الرابط: `http://localhost/PHP/GUN/Store/install.php` لتثبيت قاعدة البيانات.
3. بعد التثبيت، انتقل إلى الصفحة الرئيسية.

## بيانات الدخول (Admin)
- **اسم المستخدم**: `admin`
- **كلمة المرور**: `admin123` (يرجى التأكد من تحديث الهاش في قاعدة البيانات إذا لم يعمل، أو إنشاء مستخدم جديد وتغيير نوعه إلى admin من خلال phpMyAdmin).
*ملاحظة: السكريبت الحالي يقوم بإدراج هاش افتراضي. يفضل تسجيل حساب جديد ثم تغيير `user_type` إلى `admin` يدوياً في قاعدة البيانات.*

## المميزات
- **نظام تسجيل ودخول**: حماية كلمات المرور وتفعيل الجلسات.
- **إدارة المنتجات**: إضافة، تعديل، حذف، رفع صور.
- **سلة مشتريات**: تفاعلية مع تحديث فوري (AJAX).
- **إدارة الطلبات**: تتبع حالة الطلبات من لوحة التحكم.
- **تصميم عصري**: ثيم داكن (Dark Mode) متجاوب مع جميع الشاشات.

## التقنيات المستخدمة
- Backend: Native PHP with PDO
- Frontend: HTML5, CSS3, Vanilla JS
- Database: MySQL
- Fonts: Cairo (Arabic), Poppins (English)
